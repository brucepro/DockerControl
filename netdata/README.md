# Netdata

Real-time performance monitoring for systems and applications.

## Features
- Real-time metrics
- Health alarms
- Web dashboard
- System and app monitoring

## Quick Start
1. Start Netdata:
   ```bash
   docker-compose up -d
   ```
2. Access the web UI at [http://localhost:4022](http://localhost:4022)

## Configuration
- Config: `./config`
- Data: `./data`
