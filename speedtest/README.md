# Speedtest

Self-hosted speed test for your network.

## Features
- Test download/upload speeds
- Web UI
- History and stats

## Quick Start
1. Start Speedtest:
   ```bash
   docker-compose up -d
   ```
2. Access the web UI at [http://localhost:4038](http://localhost:4038)
