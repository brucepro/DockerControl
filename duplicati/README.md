# Duplicati

Backup software to store encrypted, compressed backups online or locally.

## Features
- Encrypted backups
- Web UI
- Scheduling
- Multiple storage backends

## Quick Start
1. Start Duplicati:
   ```bash
   docker-compose up -d
   ```
2. Access the web UI at [http://localhost:4018](http://localhost:4018)

## Configuration
- Config: `./config`
- Source: `./storage`
