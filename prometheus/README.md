# Prometheus

Open-source systems monitoring and alerting toolkit.

## Features
- Time series database
- Powerful query language (PromQL)
- Alerting rules
- Integrates with Grafana

## Quick Start
1. Start Prometheus:
   ```bash
   docker-compose up -d
   ```
2. Access the web UI at [http://localhost:4021](http://localhost:4021)

## Configuration
- Config: `./config`
- Data: `./data`
