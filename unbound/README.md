# Unbound

Validating, recursive, caching DNS resolver.

## Features
- DNS over TLS/HTTPS
- DNSSEC validation
- Caching

## Quick Start
1. Start Unbound:
   ```bash
   docker-compose up -d
   ```
2. Place config in `./config`
