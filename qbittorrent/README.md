# qBittorrent

A free and reliable BitTorrent client with a web interface.

## Features
- Web-based interface
- Torrent management
- Download scheduling
- Bandwidth control
- RSS feed support
- Search plugins

## Quick Start
1. Start qBittorrent:
   ```bash
   docker-compose up -d
   ```
2. Access the web UI at [http://localhost:4041](http://localhost:4041)
3. Default credentials: `admin` / `adminadmin`

## Configuration
- Config: `./config`
- Downloads: `./downloads`
