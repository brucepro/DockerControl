# WireGuard

Modern, fast, and secure VPN tunnel.

## Features
- Simple configuration
- High performance
- Strong encryption

## Quick Start
1. Start WireGuard:
   ```bash
   docker-compose up -d
   ```
2. Configure clients using generated configs in `./config`
