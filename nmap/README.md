# Nmap

Network scanner for security auditing and network discovery.

## Features
- Port scanning
- Service and OS detection
- Scriptable interaction

## Quick Start
1. Start Nmap:
   ```bash
   docker-compose up -d
   ```
2. Place scan results in `./scans`
